# ⚙️ Scalable Web App — Backend (Node.js + Express + MongoDB)

This is the **backend** for the Scalable Web App.  
It provides APIs for authentication (JWT), user profiles, and CRUD operations on notes.

---

## 🚀 Features
- ✅ User signup/login with **JWT**
- ✅ Passwords hashed with **bcrypt**
- ✅ CRUD operations on **Notes**
- ✅ MongoDB database connection (Mongoose)
- ✅ Centralized error handling & validation
- ✅ CORS enabled for frontend access

---

## 🧩 Tech Stack

| Tool | Purpose |
|------|----------|
| Node.js + Express | Web server & API routes |
| MongoDB + Mongoose | Database & models |
| JWT | Authentication tokens |
| bcrypt | Password hashing |
| dotenv | Environment variables |
| cors | Cross-origin requests |
